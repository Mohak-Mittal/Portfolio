// About This Project

This project is a car checkpoint driving game developed using Unreal Engine. The objective of the game is to drive through checkpoints before the timer runs out. Each checkpoint rewards the player with additional time, allowing them to continue progressing through the map.

// Tools And Engine

Unreal Engine 5 used for developing the game, implementing gameplay mechanics, and building the environment. Blueprint Visual Scripting used to create gameplay logic. Blender used to create 3D building models. FBX Asset Import Pipeline used to export models from Blender into Unreal Engine.

// Current Development Progress

Car Driving System Implemented. Timer System Working. Checkpoint System Implemented. Time Reward Mechanic Added (+10s per checkpoint). Basic Environment Created. Asset Import from Blender complete.

// Remaining Development

Expand the Environment. Add More Checkpoints. Improve Visual Quality. Optimize Performance. Gameplay Testing & Balancing.
